

# Data Visualization – Examen (Python / Plotly)

## Objectif du projet
Ce projet a pour objectif de transformer un dataset de données démographiques en visualisation interactive afin d’analyser la répartition de la population mondiale.

L’objectif est de démontrer la capacité à manipuler des données JSON et à les représenter de manière claire et interactive.

---

##  Stack technique
- JavaScript
- Plotly.js
- HTML / CSS
- JSON (dataset)
- Fetch API

---

##  Description du projet

Le projet consiste à :

- importer un dataset JSON contenant les pays et leurs populations
- trier les données par population
- sélectionner les 30 pays les plus peuplés
- créer une visualisation interactive en bar chart
- afficher les résultats dans une page web

---

##  Résultat

La visualisation permet de :
- comparer rapidement la population des pays
- identifier les pays les plus peuplés au monde
- explorer les données de manière interactive
